# Raynaldo's dotfiles

These dotfiles are managed using [chezmoi](https://www.chezmoi.io/)

## Installation

Install chezmoi and pull from github
```
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply raaynaldo
```

How to edit
```
chezmoi edit
```

Don't forget to apply after edit
```
chezmoi apply
```
